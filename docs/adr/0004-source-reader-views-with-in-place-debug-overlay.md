# Source View and Reader View are peers

Academic Reader presents the Source View and Reader View as peer views, side-by-side on larger screens and switchable on small screens. Debug information is shown in place with a Debug Overlay on the views themselves rather than through a separate inspector panel, because Academic Reader should make document structure visible without pulling the reader away from the Document. When the Debug Overlay is enabled, every eligible Block should have an always-visible colored region over the same area a browser inspector would highlight for the underlying Block or Source Geometry. Detailed Block Evidence should appear in zero-layout overlay chrome on hover or focus. Debug Overlay chrome must not participate in Source View or Reader View layout.
